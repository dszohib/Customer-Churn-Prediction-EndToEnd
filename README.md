# 📊 Customer Churn Prediction (End-to-End ML Project)

## 🚀 Project Overview
This project predicts whether a customer is likely to churn (leave the service) using Machine Learning techniques.

Customer churn prediction helps telecom and subscription businesses reduce revenue loss by identifying high-risk customers.

---

## 📌 Dataset
Dataset used: **Telco Customer Churn Dataset**
Source: Kaggle / IBM Sample Dataset

---

## ⚙️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Streamlit
- Joblib
